NAME
       mkdir - create directories

SYNOPSIS
       mkdir DIRECTORY...

DESCRIPTION
       Create directories. Parent directories must already exist.

EXAMPLES
       Create a directory:

           mkdir mydir

       Create multiple directories:

           mkdir dir1 dir2 dir3

SEE ALSO
       rm(1), touch(1)
