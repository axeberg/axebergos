NAME
       sort - sort lines of text

SYNOPSIS
       sort [-rn] [FILE...]

DESCRIPTION
       Sort lines of text. If no files are given, reads from standard input.

OPTIONS
       -r
           Reverse the result of comparisons.

       -n
           Sort numerically.

EXAMPLES
       Sort a file:

           sort file.txt

       Reverse sort:

           sort -r file.txt

       Numeric sort:

           sort -n numbers.txt
