NAME
       echo - display a line of text

SYNOPSIS
       echo [-ne] [STRING...]

DESCRIPTION
       Print arguments to standard output followed by a newline.

OPTIONS
       -n
           Do not print trailing newline.

       -e
           Enable interpretation of backslash escapes: \n (newline), \t (tab),
           \\ (backslash).

EXAMPLES
       Print text:

           echo hello world

       No trailing newline:

           echo -n "prompt: "

       With escapes:

           echo -e "line1\nline2"
