NAME
       touch - create files or update timestamps

SYNOPSIS
       touch FILE...

DESCRIPTION
       Create empty files or update modification timestamps of existing files.

EXAMPLES
       Create an empty file:

           touch newfile.txt

       Update timestamp:

           touch existing.txt

SEE ALSO
       mkdir(1), rm(1)
