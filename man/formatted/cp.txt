NAME
       cp - copy files

SYNOPSIS
       cp SOURCE DEST

DESCRIPTION
       Copy SOURCE to DEST. Does not support copying directories.

EXAMPLES
       Copy a file:

           cp original.txt backup.txt

       Copy to another directory:

           cp file.txt /home/user/documents/

SEE ALSO
       mv(1), rm(1)
