NAME
       rev - reverse lines

SYNOPSIS
       rev [FILE]

DESCRIPTION
       Reverse the characters in each line of input.

ARGUMENTS
       FILE
           File to read (or stdin if omitted).

OPTIONS
       -h, --help
           Display usage information and exit.

EXAMPLES
       Reverse a string:

           echo "hello" | rev

       Result: olleh

       Reverse file contents:

           rev file.txt

SEE ALSO
       cat(1), tr(1)
